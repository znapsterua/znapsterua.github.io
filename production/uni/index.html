О Кафедре

Обучение проводится по учебным программам которые разработаны corp. Microsoft & Oracle.

Международная сертификация

Подготовка к сдаче экзаменов и получение международных сертификатов Microsoft & Oracle на кафедре.

Microsoft IT-Academy

В соответствии с договором с корпорацией Microsoft кафедра включена в программу Microsoft IT Academy. Программа Microsoft IT Academy – это глобальное решение в области ИТ образования, которое позволяет учебным заведениям проводить высококачественное обучение с целью предоставить студентам необходимые знания и навыки для построения успешной профессиональной карьеры и жизни в информационном обществе.
